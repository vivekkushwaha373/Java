collection core package : java.util


Collection vs Collections

collection: super interface for List,set and Queue.

collections: utility / helper class with utility methods



Iteration types
1 Enumerator = works for only legacy collections class (can't remove only traverse)
2 Iterator = works for all collections classes (traverse+remove)
3 ListIterator = works for list type collections class (back-forward tranverse + remove, add, update)
