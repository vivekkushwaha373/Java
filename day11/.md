collection core package : java.util


Collection vs Collections

collection: super interface for List,set and Queue.

collections: utility / helper class with utility methods



Iteration types
1 Enumerator = works for only legacy collections class (can't remove only traverse)
2 Iterator = works for all collections classes (traverse+remove)
3 ListIterator = works for list type collections class (back-forward tranverse + remove, add, update)

## =============================== BASIC Flow ==========================================================

Iterable (interface)
   |
   └── Collection (interface)
         |
         ├── List (interface)
         │     ├── ArrayList (class)
         │     ├── LinkedList (class)
         │     └── Vector (class)
         │           └── Stack (class)
         │
         ├── Set (interface)
         │     ├── HashSet (class)
         │     │      └── LinkedHashSet (class)
         │     └── SortedSet (interface)
         │            └── NavigableSet (interface)
         │                   └── TreeSet (class)
         │
         └── Queue (interface)
               ├── PriorityQueue (class)
               └── Deque (interface)
                      ├── ArrayDeque (class)
                      └── LinkedList (class)




Map (interface)   ← separate hierarchy
   |
   ├── HashMap (class)
   │     └── LinkedHashMap (class)
   │
   ├── SortedMap (interface)
   │     └── NavigableMap (interface)
   │            └── TreeMap (class)
   │
   ├── Hashtable (class)
   │     └── Properties (class)



## ======================================== COMPREHENSIVE Flow ==============================================

Iterable (interface)
   |
   └── Collection (interface)
         |
         ├── List (interface)
         │     ├── ArrayList (class)
         │     ├── LinkedList (class)
         │     ├── Vector (class)
         │     │     └── Stack (class)
         │     └── CopyOnWriteArrayList (class)   ← concurrent
         │
         ├── Set (interface)
         │     ├── HashSet (class)
         │     │     └── LinkedHashSet (class)
         │     ├── SortedSet (interface)
         │     │     └── NavigableSet (interface)
         │     │           └── TreeSet (class)
         │     ├── EnumSet (class)
         │     └── CopyOnWriteArraySet (class)   ← concurrent
         │
         └── Queue (interface)
               ├── PriorityQueue (class)
               ├── Deque (interface)
               │     ├── ArrayDeque (class)
               │     └── LinkedList (class)
               │
               └── BlockingQueue (interface)   ← concurrent
                     ├── ArrayBlockingQueue (class)
                     ├── LinkedBlockingQueue (class)
                     ├── PriorityBlockingQueue (class)
                     ├── DelayQueue (class)
                     └── SynchronousQueue (class)

Map (interface)
   |
   ├── HashMap (class)
   │     ├── LinkedHashMap (class)
   │     └── WeakHashMap (class)
   │
   ├── IdentityHashMap (class)
   │
   ├── EnumMap (class) 
   │
   ├── SortedMap (interface)
   │     └── NavigableMap (interface)
   │            ├── TreeMap (class)
   │            └── ConcurrentSkipListMap (class)
   │
   ├── Hashtable (class)
   │     └── Properties (class)
   │
   └── ConcurrentMap (interface)
         ├── ConcurrentHashMap (class) // thread safe aka concurrent version of hashmap
         └── ConcurrentSkipListMap (class)   // thread safe aka concurrent version of treemap
