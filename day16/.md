whenever you create Enum in Java like enum direction {NORTH, SOUTH}
hava converts it into a class like class enum extends Enum{}
each constant in enum is static and final and are actually objects of class direction
hence 

class direction {
    
    //java also created a private class as well to prevent object creation outside this class
    private direction(){

    }

    public static final Direction NORTH = new Direction();
    public static final Direction SOUTH = new Direction();
}
