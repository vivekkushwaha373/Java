> for classes we have only two modifiers: public and default
> one file belongs to same package can only have one public class
> for member access modifires are--> public, private, protected, default
> protected member can be accessed within same package as well as in subclass of other package if class is public
> default member can be accessed within same package anywhere but not in subclass
> private member can be only accessed within the class it is defined in not in other class be it belongs to same packge or outside package
> assesibility order [public > protected > default > private]
> abstract method can have constructor and normal methods as well 
> final varible also inherited but not initialized again
> final class can not be inherited
> An interface can contain:

Abstract methods
Default methods
Static methods
Constants (public static final fields)
